# ShareSmart Image Gallery

A gallery page that pulls images from a remote source, and display them.

## Getting Started

1. Clone this repo
2. Install npm packages
```
npm install
```
3. Run Application
```
npm start
```
4. Go to http://0.0.0.0:8080/

#### Dependencies

- react
- react-dom
- css-loader
- style-loader
- webpack
